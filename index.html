⚙ Firebase + GitHub Pages 設定

請依照 SETUP.md 完整步驟設定。簡要：

1. Firebase

  1. 到 console.firebase.google.com 建立新專案
  2. Build → Authentication → Sign-in method → 啟用 Google
  3. Build → Firestore Database → Create database（test mode）
  4. Project settings → 加入Web app，複製 firebaseConfig
  5. 編輯本 HTML 檔案，找到 FIREBASE_CONFIG 區塊，貼上你的配置
  6. Authentication → Settings → Authorized domains → 加入你的 GitHub Pages 網域

2. GitHub Pages

  1. 建立新 repo（public）
  2. 把這個檔案改名為 index.html 上傳
  3. Settings → Pages → Source: main branch / root
  4. 等 1–2 分鐘，網址為 https://<username>.github.io/<repo>/
  5. 回 Firebase 把這個網址加到 Authorized domains

Firestore 安全規則

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null
        && request.auth.uid == userId;
    }
  }
}
